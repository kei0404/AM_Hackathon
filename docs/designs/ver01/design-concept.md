# Design Concept - Data Plug Copilot Dashboard

**Version**: ver01
**Created**: 2025-01-17
**Designer**: UI/UX Design Team

---

## 概要

Data Plug Copilotのダッシュボードデザインコンセプトです。ウェブアプリとモバイルアプリの両方に対応したレスポンシブデザインを提供します。

---

## コアバリュー

**「車がその瞬間だけあなたのものになる」**

- プライバシー・バイ・デザイン原則を視覚的に表現
- セッションベースの接続状態を明確に表示
- ユーザーのパーソナルデータを中心としたUI設計

---

## デザイン原則

### 1. 明確なセッション状態表示
- 車との接続状態（未接続/接続中/終了）を常に視覚化
- プライバシー保護のステータスを明示

### 2. シンプルな意思決定サポート
- 最大3回の質問で目的地を決定するフローを反映
- 選択肢は3つ以内に限定

### 3. 運転中の安全配慮
- 大きなタップターゲット
- 高コントラストな配色
- 最小限のテキスト

### 4. パーソナライズの可視化
- お気に入りスポットの視覚的表示
- 訪問履歴のタイムライン
- 嗜好設定の直感的な操作

## コンポーネント構成

### ダッシュボード要素

1. **ヘッダー**
   - ロゴ/アプリ名
   - セッション状態インジケーター
   - ユーザーアバター

2. **クイックアクション**
   - 「今日どこに行く？」AIアシスタント起動
   - QRスキャン（車に接続）
   - グループモード起動

3. **今日のスケジュール**
   - タイムライン形式
   - 目的地カード
   - 進捗状態

4. **お気に入りスポット**
   - カルーセル/グリッド表示
   - カテゴリフィルター

5. **最近の訪問履歴**
   - リスト形式
   - 日時・滞在時間表示

6. **セッション管理**
   - 接続/切断ボタン
   - データ共有範囲選択
   - データ消去確認

## AIチャット画面 (chat-mobile.html)

### 概要

ダッシュボードの「AIに相談する」ボタンをタップした後に表示されるチャット画面です。最大3回の質問でユーザーの目的地を絞り込むフローを実現します。

### デザイン要素

1. **チャットヘッダー**
   - 戻るボタン（ダッシュボードに戻る）
   - AIアシスタントのアバターと状態表示
   - 質問進捗インジケーター（●●○ 2/3形式）

2. **メッセージエリア**
   - AIメッセージ: 左寄せ、白背景、アバター付き
   - ユーザーメッセージ: 右寄せ、青背景
   - タイムスタンプ表示
   - フェードインアニメーション

3. **クイック返信ボタン**
   - 最大3つの選択肢
   - 絵文字 + テキストで直感的に
   - タップしやすい大きなボタン（48px以上）

4. **入力エリア**
   - 音声入力ボタン（運転中の安全配慮）
   - テキスト入力フィールド
   - 送信ボタン
   - 提案バー（お気に入りから選ぶ、など）

### UXフロー

```
[ダッシュボード]
    ↓ 「AIに相談する」タップ
[チャット画面]
    ↓ AIがウェルカムメッセージ
    ↓ 質問1: 「カフェ系？自然系？」（2択）
    ↓ 質問2: 「いつもの場所？新しい場所？」（3択）
    ↓ 質問3: 具体的な候補を提示（3つ以内）
[目的地決定]
    ↓ スケジュールに追加
[ダッシュボード/ナビ開始]
```

### 運転中モード

- 音声入力を優先表示
- 選択肢は3つ以内
- 大きなタップターゲット
- 安全注意メッセージを表示可能
